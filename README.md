# Development

Run:

     npm link
     
Commands:

    tcc merge 
    tcc zip

Usage:

Put a "tcc-config.json" file in each project. First key in mapping will be used as starting branch.

## Ubuntu 16.04
before yarn / npm install:
    
    sudo apt install libssl-dev
    
# Features
    
- Zip
- Merge
    - Ask to push afterwards
